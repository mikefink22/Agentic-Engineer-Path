# Agentic Engineer: Professional Validator Dashboard 🚀

Este proyecto es una herramienta interactiva diseñada para gestionar la transición del rol de desarrollador tradicional al de Agentic Engineer (Rol Validador). El objetivo es centralizar el progreso técnico, la capacitación en arquitectura avanzada y la preparación para certificaciones Cloud (AWS), permitiendo simular el criterio técnico de un perfil Senior.

## 🎯 El Concepto de "Validador"

En el paradigma actual, la IA genera el código, pero el humano lo audita. Este dashboard permite trackear el dominio sobre:

Ingeniería de Código: Gestión de memoria, inmutabilidad y DI en .NET.

Arquitectura: Patrones Hexagonales y Clean Architecture.

Seguridad: Auditoría OWASP Top 10 aplicada a código generado por IA.

Infraestructura: Despliegue y escalabilidad en AWS.

## 🛠️ Tecnologías Utilizadas

Frontend: HTML5, Tailwind CSS (Styling), FontAwesome (Icons).

Lógica: JavaScript (Vanilla) con persistencia en localStorage.

Visualización: Chart.js para el radar de competencias dinámico.

## ✨ Características Principales

Radar de Competencias Real-Time: Visualiza tu crecimiento técnico a medida que completas tareas.

Dynamic Seniority Widget: Un panel de perfil que evoluciona de "Junior Auditor" a "Senior Agentic Engineer" según tu progreso.

Centro de Recursos Clickables: Acceso directo a documentación oficial de AWS, canales especializados de .NET y herramientas de seguridad.

Roadmap por Fases: 4 fases críticas de capacitación técnica detallada.

Estrategia de Portafolio: Checklist para aplicar Chaos Engineering y limpieza de repositorios para ganar seniority.

## 🚀 Cómo usar este Dashboard

Clona el repositorio.

Abre el archivo index.html en tu navegador.

Marca tus avances en las pestañas Roadmap Técnico y Portafolio.

Observa cómo el Dashboard actualiza tus métricas y etiquetas de nivel automáticamente.
